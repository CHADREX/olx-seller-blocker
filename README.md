# OLX Seller Blocker

Userscript для блокування продавців на OLX.ua з можливістю приховування їхніх оголошень.

## 🎯 Можливості

- **Блокування продавців** — один клік, щоб заблокувати всі оголошення автора
- **Відображення імені** — кнопка показує ім'я продавця
- **Два режими перегляду:**
  - 🔍 Показати заблоковані (напівпрозорі з міткою)
  - 👁️ Сховати заблоковані (повністю приховує `display: none`)
- **Кеш маппінгу** — швидка робота без зайвих запитів
- **localStorage** — список заблокованих зберігається між сесіями
- **Автоматичне оновлення** — всі кнопки продавця синхронізуються

## 📦 Встановлення

1. Встановіть [Tampermonkey](https://www.tampermonkey.net/) або [Violentmonkey](https://violentmonkey.github.io/)
2. Натисніть [**Встановити скрипт**](https://github.com/CHADREX/olx-seller-blocker/raw/main/OLX%20Seller%20Blocker-1.2.user.js)
3. Підтвердіть встановлення
4. Оновіть сторінку OLX

## 🚀 Використання

### Блокування продавця

1. Відкрийте будь-яку категорію на OLX.ua
2. Під кожним оголошенням з'явиться кнопка: **🚫 Заблокувати: Ім'я**
3. Натисніть кнопку → підтвердіть
4. Всі оголошення цього продавця стануть напівпрозорими

### Приховування заблокованих

1. У правому нижньому куті відкрийте панель **🚫 OLX Blocker**
2. Натисніть **🚫 Сховати заблокованих**
3. Всі заблоковані оголошення зникнуть зі сторінки
4. Щоб повернути: **👁️ Показати заблокованих**

### Розблокування

1. Знайдіть будь-яке оголошення заблокованого автора
2. Натисніть **✅ Розблокувати: Ім'я**
3. Всі його оголошення стануть знову видимими

### Управління списком

1. Панель **🚫 OLX Blocker** → **Керувати списком**
2. Перегляд усіх заблокованих з датами блокування
3. Для розблокування використовуйте кнопку на картці

## 🔧 Технічні деталі

### Збережені дані (localStorage)

- `olx_blocked_sellers` — список заблокованих продавців
- `olx_ad_seller_cache` — кеш маппінгу ID оголошення → ID продавця
- `olx_hide_blocked_cards` — стан видимості (`true`/`false`)

### Як працює

1. **Парсинг даних:**
   - Читає `window.__PRERENDERED_STATE__` на сторінці
   - Створює маппінг ID оголошення → дані продавця
   - Зберігає в localStorage кеш для швидкості

2. **Додавання кнопок:**
   - Знаходить всі картки `[data-cy="l-card"]`
   - Вставляє кнопку після `[data-testid="location-date"]`
   - Fallback через AJAX, якщо немає в кеші

3. **Перемикання видимості:**
   - Додає/видаляє клас `olx-hide-blocked` на `<body>`
   - CSS: `body.olx-hide-blocked .olx-blocked-card { display: none !important; }`

### Сумісність

- ✅ OLX.ua
- ✅ Tampermonkey
- ✅ Violentmonkey
- ✅ Chrome, Firefox, Edge, Safari (з розширенням)

## ⚙️ Налаштування

У файлі скрипта можна змінити:

```javascript
const CONFIG = {
    STORAGE_KEY: 'olx_blocked_sellers',
    HIDE_STATE_KEY: 'olx_hide_blocked_cards',
    BLOCK_BUTTON_TEXT: '🚫 Заблокувати',
    UNBLOCK_BUTTON_TEXT: '✅ Розблокувати',
    BLOCKED_MESSAGE: '⛔ Заблоковано',
    DEBUG: false // Включити логи в консолі
};
```

## 🐛 Відомі проблеми

- **React перерисовує DOM** — скрипт відновлює кнопки кожні 3 секунди
- **Кешування браузера** — при звичайному F5 можуть бути помилки, використовуйте Ctrl+Shift+R
